# 极限存在准则
## 两个重要极限
1. 数列 {x<sub>n</sub>} 和 {y<sub>n</sub>}, 
    1. 存在 n<sub>0</sub> ∈ N, 当 n > n<sub>0</sub> 时，y<sub>n</sub> ≤ x<sub>n</sub> ≤ z<sub>n</sub>
    2. lim<sub>n→∞</sub>y<sub>n</sub> == a, lim<sub>n→∞</sub> == a
    则lim<sub>n→∞</sub>x<sub>n</sub> == a
    * 对于函数定义，也适用
        1. g(x) ≤ f(x) ≤ h(x)
        2. lim g(x) == A, lim h(x) == A
