#  `range` 函数的使用
1. 只给一个参数 (默认起始为 0, 步长为 1)

    ```py
    r = range(4) # 等同于 range(0, 4)
    print(list(r)) # [0, 1, 2, 3]
    ```

2. 给定两个参数 (默认步长为 1)

    ```py
    r = range(0, 4) # [0, 1, 2, 3]
    ```

3. 给定三个参数

    ```py
    r = range(0, 5, 2) # [0, 2, 4]
    ```

## 数列的判断
```py
print (2 in r) # True    
print (0 not in r) # False
```

## 特点
* 无论整数序列有多长，其**占用的内存空间是相同的**。
